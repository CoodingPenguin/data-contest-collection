# 김해시 화재발생 예측모델 개발

> 경상남도 소방 및 건물 관련 데이터를 활용하여 김해시의 건축물의 화재 위험도 분석 및 예측 모델을 개발한다.

## 대회 개요
- `목적` : 김해시의 건물의 화제 예상 여부를 예측
- `기준` : F1-Score
- `데이터` : [경상남도 소방 및 건물 관련 데이터](https://compas.lh.or.kr/subj/past/data?subjNo=SBJ_1910_002)


## 대회 결과
F1-Score 0.45994로 최종 88등

## 대회 후기
- 컬럼이 매우 많았지만 대부분의 컬럼의 결측치가 많아서 많은 컬럼을 날려야했다.
- 첫 데이터분석 공모전이라 어떻게 전처리를 해야할지 팀원과 많이 토론을 많이 했다.
- 건물관련 지식이 없어서 구글링 및 지인들의 도움을 받아서 데이터 전처리를 진행했다.
- 타이타닉과 같은 잘 정제된 데이터를 보다 날 것의 데이터를 보니 전처리가 정말 중요하다는 것을 깨달았다.
- 마지막에 PCA를 해서 평가 지표의 값을 더 높일 수 있었으나 시간이 없어서 적용하지 못했다.
