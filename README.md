# 데이터 분석 대회 코드 모음 

그동안 참가한 **데이터 분석/아이디어 대회 코드 모음집**입니다.

## 🔥 주의 사항

- 일부 대회의 규정에 따라 데이터셋은 공개하지 않습니다.
- 아이디어 대회를 제외한 분석 대회의 코드 관련 질문은 가능합니다.
- 제 코드를 참고하는 것은 가능하지만 그대로 복붙은 불가능합니다. 
- 만약 분석 결과를 참고하고 싶으신 경우 출처를 꼭 남겨주세요.
