# 제 1회 체육종합 데이터 활용 경진대회

> 체육종합 데이터를 활용한 창의적인 아이디어 및 새로운 비즈니스 모델 발굴한다.

## 준비 방향
국민체육진흥공단의 [국민체력측정 현황 데이터](https://www.culture.go.kr/bigdata/user/data_market/detail.do?id=c99df919-f2c9-4ceb-999d-82688c028031)를 가지고 국민체력100의 출장 서비스 기획

## 대회 결과
미정

## 대회 후기
- 경정, 경마 등을 제외한 국민체력100관련 데이터 중 온전한 데이터는 [국민체력측정 현황 데이터](https://www.culture.go.kr/bigdata/user/data_market/detail.do?id=c99df919-f2c9-4ceb-999d-82688c028031) 정도 밖에 없었다.
- 거기다 해당 데이터의 이용자 관련 정보가 너무 부족해서 데이터분석 시 어려움이 있었다.
- 첫 대회인 만큼 관련 자료가 부족해서 대회 준비의 방향성을 잡느라 힘들었다.
- 모델보다는 전처리 및 EDA에 중점을 두었는데 코드를 따라 치다가 직접 EDA를 하려니 어려웠다. 특히 새로운 데이터프레임을 만들고 통합하는 과정이 제일 힘들었다.
